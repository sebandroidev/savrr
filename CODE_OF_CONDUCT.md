# Code of conduct

## The short version

Be decent to each other. This is a small project maintained in people's spare time. Assume good faith, keep criticism about the code rather than the person, and remember there's a human on the other end of the thread.

## What's expected

- Give and take feedback gracefully. A rejected patch or a "no" on a feature isn't a personal slight.
- Respect that maintainers volunteer their time and don't owe anyone a fix on a deadline.
- Keep discussion focused on the work.

## What's not okay

Harassment, insults, discrimination, sexualized language or attention, publishing someone's private information, and any other conduct that a reasonable person would find hostile or unwelcome. This applies in issues, pull requests, commit messages, and anywhere else the project is discussed.

## If something goes wrong

Report it to **nssoftdev@gmail.com**. Reports stay private. The maintainer will look into it and can warn, block, or ban anyone whose behavior is a problem, and can remove or edit contributions that break these rules.

## Attribution

This is adapted from the spirit of the [Contributor Covenant](https://www.contributor-covenant.org), trimmed to fit a project this size.
